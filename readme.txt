Natty Parks is a website that uses the NPS API to display information about the parks stored in the database.

The functionality is selecting the desired state which will accumulate the National Parks within that state. It's also populating historic sites, monuments, etc ontop of parks.

As a user, I can select a state from the dropdown to view the parks within the state

As a user, I can click the "EXPLORE" button to view a random park.

As a user, I can hover over the cards to see additional information.

As a user, I can click the website button to view more information about the park.
